# INTERVIEW SCHEDULER
Single page application using **React**.
Allows the user to book an appointment with an interview.

Testing frameworks used: **jest** and **storybook**.

# ```DEPENDENCIES```

- axios
- classnames
- react
- react-dom
- react-scripts
